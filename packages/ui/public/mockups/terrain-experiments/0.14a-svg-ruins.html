0.14a — SVG Ruins Test

Hand-drawn SVG terrain pieces inspired by the reference image. Goal: see if pure SVG can produce acceptable grimdark ruins.

Reference

L-shaped ruin with: stone block walls, jagged broken edges with rubble, bullet holes/craters, pillars, imperial eagle motifs, structural beams visible in broken sections. Transparent background.

Wall Section with Arch

Straight wall segment with arched doorway — like the overhang in the reference

Notes

  • All pieces use SVG patterns for stone blocks and floor tiles — no noise/random textures
  • Broken edges are manually defined jaggy paths with rubble polygons along them
  • Bullet holes are concentric circles (outer ring + dark center)
  • Imperial eagles are simplified SVG paths
  • Transparent background (checkerboard shown via CSS only)
  • All elements are crisp geometry — no filters/blur (keeps it clean for game overlay)
