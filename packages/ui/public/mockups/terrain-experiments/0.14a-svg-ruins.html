0.14a — SVG Ruins: SOTA Filters + Floor/Roof Layers

18 terrain pieces (6 types × 3 variations) with advanced SVG filters: feTurbulence, feDiffuseLighting, feSpecularLighting, feDisplacementMap, feColorMatrix. Each piece has toggleable Floor and Roof layers.

SVG Filter Techniques Used

  • feTurbulence + feDiffuseLighting: Perlin noise bump map with directional lighting creates realistic 3D stone texture
  • feSpecularLighting: Wet/metallic highlights on stone surfaces for grimdark industrial look
  • feColorMatrix: Greenish grime weathering overlay shifts colors for decay
  • feDisplacementMap: Turbulence displaces wall edges for organic irregularity
  • SVG Patterns: Stone block and floor tile patterns with mortar lines
  • Composite filter chains: Base color → stone grain → lighting → weathering → damage
  • Floor/Roof layer separation: Floor renders below units, Roof (walls + ceiling fragments) renders above
